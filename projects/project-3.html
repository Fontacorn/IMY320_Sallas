FLIPP'N BURGERS GAME
2022
Stewart Coetzee &
Lakshita Padiachey

For this project I was trying to achieve a website where users can learn tips on how to get your cat to sleep peacefully at night so that the user can too. Users can interactively perform tasks which they should try in real life. Users also should be able to learn about how these tips help solve the problem. I have included 4 tasks that you could possibly do to help. The website’s style and information are based off an infographic.
